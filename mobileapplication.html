3. Application Development Type
  • Type: Cross Platform
  • Advantages: Supports native user interfaces and therefore can provide the type of user experience that iOS and Android users are accustomed to.
    Developer only needs to be fluent in .NET/C# (That is only one Programming Language)
  • Disadvantages: Lack of ability to share code outside of Xamarin
    Lack of community support for Xamarin.

    10. Features Xamarin is a platform consisting of Xamarin.iOS, Xamarin.Android, Xamarin.Mac, Xamarin Test Cloud, and HockeyApp.
      Xamarin.iOS: Use for IOS app development.
      Xamarin.Android: Use for Android app development.
      Xamarin.Mac: Use for MAC platform app development.
      Xamarin Test Cloud: Use for testing of Xamarin code on a various virtual mobile platform
    11. Debugger Visual Studio for Mac has a native debugger allowing debugging support for Xamarin.iOS, Xamarin.Mac, and Xamarin.Android applications. Visual Studio for Mac uses the Mono Soft Debugger, which is implemented into the Mono runtime, allowing Visual Studio for Mac to debug managed code across all platforms.
    12. Built-In Emulator Xamarin does not comes Built-In in any Operating System. But as it is an Opensource while installing Visual Studio, you can include xamarin with it.

    20. Third Party Emulator Genymotion is a fast third-party emulator that can be used instead of the default Android emulator. It runs as a virtual machine on your computer, so you get much better performance, and it's much more like using a real device instead of the Android Emulator. You can download and install the Genymotion Personal Edition to explore the possibilities of this tool. Payed plans are available with more features. Because it runs as a virtual machine you also need to install the free VirtualBox from Oracle.
